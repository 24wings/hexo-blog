HTTP头信息

  • UserAgent 用户操作系统,浏览器,浏览器内核以及版本号
  • Referer: 从哪个页面点过来
  • Cookie: Cookie会话追踪
  • Server: 服务器信息
  • X-powered-By 编程语言和框架
  • Content-length: 响应体的长度
  • Content-type 内容类型
